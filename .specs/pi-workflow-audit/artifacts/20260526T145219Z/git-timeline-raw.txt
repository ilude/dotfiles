    feat(commands): add /optimize-skill for academic-grounded skill improvement
    
    New slash command that improves skills by grounding them in academic
    research while preserving practical experience.
    
    5-phase process:
    1. Understand (sonnet) - Extract problem domain and search terms
    2. Research (sonnet, parallel) - Search academic literature
    3. Synthesize (opus) - Map theory to practice
    4. Restructure (opus) - Rewrite with academic grounding
    5. Apply - User confirms changes
    
    Key features:
    - Parallel research agents for comprehensive literature search
    - Theory-to-practice mapping table
    - Preservation rules to keep practical examples/anti-patterns
    - Structured template for consistent skill format
    - Sources section with academic and industry references
    
    Motivated by successful code-review skill optimization that used
    MUST vs MAY analysis theory to eliminate false positives.
