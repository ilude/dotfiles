# Research Vault Index

This vault is a context pool for research notes across multiple topics. It is not
a roadmap or commitment to build everything captured here.

## Vault guidance

- [[README]] — human entry point
- [[AGENTS]] — vault-wide agent guidance

## Topics

- [[agent-workflows/index]] — agent workflow tools, patterns, projects, videos,
  and KISS ideas applicable to Pi/dotfiles.

## Topic convention

Top-level folders represent broad research topics. A topic should usually contain:

- `README.md` — quick human entry point
- `index.md` — curated map of notes
- optional `AGENTS.md` — topic-specific guidance when vault-wide rules are not
  enough

Prefer a small, useful topic folder over a perfect taxonomy.
